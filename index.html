Technical Deep Dive

Technology Stack

  • Rust: High-performance, memory-safe backend with concurrency for efficient processing.
  • Slint: Responsive, cross-platform GUI for intuitive user interaction.
  • Tokio: Asynchronous scraping and task management for fast data retrieval.
  • Thirtyfour: Reliable WebDriver automation for CUSIS interaction.
  • Consumer-Producer Threading: Thread-safe communication between Slint frontend and Rust backend using Rustâ€™s channels.
  • Geckodriver & Firefox: Robust web scraping with automated setup.

Design Decisions

Single WebDriver: Chosen to minimize resource usage and simplify session management, ensuring efficiency for a single-user desktop app.

CLI to GUI Transition: Evolved from a CLI tool to a Slint-based GUI to enhance accessibility for all CUHK students.

Scheduling Algorithm

Employs recursive backtracking with scoring metric to generate optimized, conflict-free schedules based on user preferences.

Security

Runs locally, feeding credentials directly to the CUSIS portal without storage or server-side processing, ensuring user privacy and security.
